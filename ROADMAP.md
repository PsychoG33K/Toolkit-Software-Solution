# Roadmap

## V1.0 Priorities

- Controlled distribution of CockpitDiag Enterprise V1.0.
- Trial licence workflow documentation.
- Clean release ZIP with SHA256 checksum.
- Client-side extraction and licence validation test.

## V1.1 Candidates

- Authenticode or Azure Trusted Signing.
- PortableMode for logs and runtime data stored relative to executable.
- Stronger local anti-clock-rollback check for offline trials.
- Improved System Health hybrid scoring based on diagnostic reports.
- AutoDiag energy/sleep diagnostics.

## V2 Candidates

- Server-side activation and licence verification.
- Customer portal or controlled licence delivery workflow.
- Two-step AI diagnostic workflow with targeted questions before final verdict.
- Optional external stress-test log import.
- Optional anonymous telemetry/data contribution with explicit consent.
- Installer and update channel.

# Installation - MotherDiag Technician Edition

## Distribution

MotherDiag Technician Edition est distribuée via un canal officiel Toolkit Software Solution, sur demande.

Le lien de téléchargement et les informations de vérification SHA256 sont communiqués directement aux utilisateurs autorisés.

Le ZIP n'est pas stocké directement dans ce dépôt GitHub.

## Avant Installation

Utilisez uniquement des paquets provenant d'un canal officiel Toolkit Software Solution.

Avant exécution, vérifiez :

- que le paquet provient d'une source officielle ;
- que le fichier n'a pas été renommé ou modifié ;
- que les conditions de licence correspondent à l'édition fournie ;
- que les données sensibles ne sont pas partagées dans des rapports ou prompts IA sans anonymisation.

## Licence

MotherDiag Technician Edition correspond à l'édition technicien / freemium-pro.

Selon le canal et la version fournie, certaines fonctions peuvent être limitées ou soumises à une licence.

Ne publiez jamais un fichier `license.json`, une licence client, un rapport complet, un dump mémoire ou un log brut dans un espace public.

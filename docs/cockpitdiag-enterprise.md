# CockpitDiag Enterprise

CockpitDiag Enterprise is a professional Windows diagnostic and maintenance suite for advanced troubleshooting, customer support and controlled technical interventions.

## Main Areas

- Maintenance and Windows repair workflows
- Network diagnostics and DNS tools
- Hardware, drivers and storage checks
- Services and startup review
- Debug and crash-report assistance
- Portable external-tool launcher, with tools supplied by the user
- AutoDiag report generation with manual AI prompt workflow

## Licensing

CockpitDiag Enterprise requires a valid `license.json` for protected functions. The licence is validated locally through the packaged licensing component.

A public/demo ZIP should not contain `license.json`. Trial or customer licences are delivered separately.

## AI Workflow

CockpitDiag Enterprise can prepare a local AI prompt and open an external AI provider. No automatic transmission is performed in the V1 workflow.

The user uses their own AI account, API access or provider account.

## Distribution Status

V1.0 is distributed as a controlled release candidate, unsigned Authenticode for now, with SHA256 checksum verification.

# FAQ

## CockpitDiag Enterprise Est-Il Open Source ?

Non. Ce dépôt est une vitrine publique de release et de vérification. Il ne contient pas le code source propriétaire de CockpitDiag Enterprise.

## Pourquoi Le ZIP N'Est-Il Pas Dans Le Dépôt ?

Le ZIP client est hébergé sur le canal Toolkit Software Solution. Le dépôt public contient les informations de release, le SHA256 et les documents utiles.

## À Quoi Sert Le SHA256 ?

Le SHA256 est une empreinte d'intégrité. Il permet de vérifier que le fichier téléchargé est exactement celui publié par Toolkit Software Solution.

Il ne chiffre pas le fichier et ne contrôle pas l'accès au téléchargement.

## Le Lien De Téléchargement Est-Il Public ?

Le lien est non indexé, mais toute personne qui possède le lien peut télécharger le ZIP. La licence d'activation reste fournie séparément.

## Pourquoi `license.json` Est-Il Séparé ?

Le ZIP distribue le logiciel. Le fichier `license.json` active les droits accordés à un client ou testeur autorisé. Le séparer évite de publier une licence client dans un paquet partageable.

## Puis-Je Partager Ma Licence ?

Non. Il est interdit de partager, publier, revendre ou transférer un fichier `license.json` sans autorisation écrite de Toolkit Software Solution.

## CockpitDiag Envoie-T-Il Des Données À Une IA ?

Non. CockpitDiag peut générer un prompt anonymisé, mais l'utilisateur décide ensuite de le copier ou non dans son propre compte IA.

## Toolkit Software Solution Fournit-Il Des Crédits IA ?

Non. Toolkit Software Solution ne fournit, ne revend et ne sous-licencie aucun crédit, compte, token, clé API ou accès API de fournisseur IA tiers.

## Faut-Il Lancer CockpitDiag En Administrateur ?

Oui, c'est recommandé. Certaines fonctions de diagnostic et maintenance Windows demandent une élévation administrateur.

## Que Faire Si La Licence Ne Fonctionne Pas ?

Vérifiez que le fichier s'appelle exactement `license.json`, qu'il est placé à côté de `CockpitDiag.exe`, que le ZIP est extrait, puis relancez en administrateur. Si le problème persiste, contactez contact@toolkitsoftware.tech.

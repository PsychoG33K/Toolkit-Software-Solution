# FAQ

## CockpitDiag Enterprise Est-Il Open Source ?

Non. Ce depot est une vitrine publique de release et de verification. Il ne contient pas le code source proprietaire de CockpitDiag Enterprise.

## Pourquoi Le ZIP N'Est-Il Pas Dans Le Depot ?

Le ZIP client est heberge sur le canal Toolkit Software Solution. Le depot public contient les informations de release, le SHA256 et les documents utiles.

## A Quoi Sert Le SHA256 ?

Le SHA256 est une empreinte d'integrite. Il permet de verifier que le fichier telecharge est exactement celui publie par Toolkit Software Solution.

Il ne chiffre pas le fichier et ne controle pas l'acces au telechargement.

## Le Lien De Telechargement Est-Il Public ?

Le lien est non indexe, mais toute personne qui possede le lien peut telecharger le ZIP. La licence d'activation reste fournie separement.

## Pourquoi `license.json` Est-Il Separe ?

Le ZIP distribue le logiciel. Le fichier `license.json` active les droits accordes a un client ou testeur autorise. Le separer evite de publier une licence client dans un paquet partageable.

## Puis-Je Partager Ma Licence ?

Non. Il est interdit de partager, publier, revendre ou transferer un fichier `license.json` sans autorisation ecrite de Toolkit Software Solution.

## CockpitDiag Envoie-T-Il Des Donnees A Une IA ?

Non. CockpitDiag peut generer un prompt anonymise, mais l'utilisateur decide ensuite de le copier ou non dans son propre compte IA.

## Toolkit Software Solution Fournit-Il Des Credits IA ?

Non. Toolkit Software Solution ne fournit, ne revend et ne sous-licencie aucun credit, compte, token, cle API ou acces API de fournisseur IA tiers.

## Faut-Il Lancer CockpitDiag En Administrateur ?

Oui, c'est recommande. Certaines fonctions de diagnostic et maintenance Windows demandent une elevation administrateur.

## Que Faire Si La Licence Ne Fonctionne Pas ?

Verifiez que le fichier s'appelle exactement `license.json`, qu'il est place a cote de `CockpitDiag.exe`, que le ZIP est extrait, puis relancez en administrateur. Si le probleme persiste, contactez contact@toolkitsoftware.tech.
